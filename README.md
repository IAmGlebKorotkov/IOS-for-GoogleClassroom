## Требования

- Mac с macOS 13 или новее
- [Xcode](https://apps.apple.com/app/xcode/id497799835) (бесплатно в Mac App Store)
- iPhone (опционально — можно использовать встроенный симулятор)

---

## Запуск

### 1. Откройте проект

Запустите файл проекта в Xcode:

```
IOS-for-GoogleClassroom-main/GoogleClassRoom.xcodeproj
```

Xcode откроет и настроит проект автоматически.

---

### 2. Настройте сервер

По умолчанию приложение подключается к серверу:

```
http://37.21.130.4:5000
```

Чтобы изменить адрес сервера, откройте файл:

```
IOS-for-GoogleClassroom-main/GoogleClassRoom/Network/APIClient.swift
```

И замените значение `baseURL` на ваш адрес.

---

### 3. Запустите приложение

**На симуляторе** — выберите устройство в панели Xcode и нажмите **⌘ + R**.

**На iPhone** — подключите телефон к Mac по USB, выберите его в списке устройств и нажмите **⌘ + R**. При первом запуске потребуется войти в Apple ID через **Xcode → Settings → Accounts**.

---

## Возможные проблемы

| Проблема | Решение |
|---|---|
| Ошибка «Untrusted Developer» на iPhone | **Настройки → Основные → VPN и управление устройством** → доверять сертификату |
| Ошибка сборки | **Product → Clean Build Folder** (⇧⌘K), затем повторный запуск |
| Приложение не подключается к серверу | Проверьте адрес в `APIClient.swift` и доступность сервера |
